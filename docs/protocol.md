## RUC Protocol
```
 0               1               2                
 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7
+---------------+---------------+---------------+
|   Source ID   |   Reserverd   | Message Type  |
+---------------+---------------+---------------+
|   Reserved    |       Data Length (BE)        |
+---------------+-------------------------------+
|                                               |
|                 Data (0-n Bytes)              |
|                                               |
+-----------------------------------------------+
```
